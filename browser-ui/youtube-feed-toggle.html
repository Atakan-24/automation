▶ YouTube Feed Toggle

Brave iOS • Home Feed ein/ausschalten

🔖 Methode 1 — Bookmarklet Empfohlen

Speichere das Bookmarklet in Brave. Tippe es auf YouTube an — der Home Feed verschwindet. Nochmal tippen: Feed erscheint wieder.

  1. Kopiere den Code unten.
  2. Tippe in Brave auf das Lesezeichen-Symbol → Lesezeichen hinzufügen.
  3. Lösche die URL komplett und füge den kopierten Code ein.
  4. Namen vergeben z.B. „YT Feed Toggle“ → Speichern.
  5. Auf YouTube öffnen → Lesezeichen antippen → Feed weg!
javascript:(function(){var s=document.getElementById('__ytFeedToggle');if(s){s.remove();}else{s=document.createElement('style');s.id='__ytFeedToggle';s.textContent='ytd-rich-grid-renderer,ytd-browse[page-subtype="home"] #contents.ytd-browse,ytd-browse[page-subtype="home"] #secondary.ytd-browse{display:none!important}';document.head.appendChild(s);}})();

🛡 Methode 2 — Brave Filter Liste Dauerhaft aus

Füge die Filter-URL in Brave ein. Der Home Feed ist dann immer ausgeblendet — ohne Tippen.

  1. Kopiere die URL unten.
  2. Brave → Einstellungen → Brave Shields & Datenschutz → Inhaltsfilterung.
  3. Ganz unten: Benutzerdefinierte Filterliste hinzufügen.
  4. URL einfügen → Bestätigen.
